# Zitate - Quote Collection App

An iOS application for collecting and managing sayings, citations, and "bon mots" with rich metadata including location, media, and intelligent organization capabilities.

## Overview

Zitate allows users to:
- Capture quotes with automatic location tagging
- Attach images and audio recordings
- Organize quotes with authors and labels
- Create smart folders with dynamic criteria
- Search and browse their collection

## Requirements

- iOS 17.0+
- Xcode 15.0+
- Swift 5.9+

## Project Structure

```
Zitate/
├── Models/              # Core Data models
├── ViewModels/          # MVVM view models
├── Views/              # SwiftUI views
├── Services/           # Business logic services
├── Utilities/          # Helper utilities
└── Resources/          # Assets and Core Data model
```

## Architecture

The app follows the MVVM (Model-View-ViewModel) pattern:
- **Models**: Core Data entities (Entry, Author, Label, etc.)
- **Views**: SwiftUI views for UI
- **ViewModels**: ObservableObject classes managing state
- **Services**: Singleton services (PersistenceController, LocationService)

### Features Implemented

1. **Data Models**
   - Entry (quote text, location, timestamps)
   - Author (name, biographical data)
   - Label (tags for categorization)
   - ImageAttachment and AudioAttachment
   - SmartFolder (dynamic query-based folders)

2. **Services**
   - PersistenceController (Core Data stack)
   - LocationService (GPS and geocoding)

3. **Entry Creation**
   - Multi-line text input with character counter (1-10,000 chars)
   - Automatic location capture and display
   - Address reverse geocoding
   - Save/Cancel functionality

## Setup Instructions

### 1. Open the Project

Since this is a manual project structure (not generated by Xcode), you'll need to:

1. Open Xcode
2. Create a new iOS App project with:
   - Product Name: Zitate
   - Organization Identifier: com.yourcompany
   - Interface: SwiftUI
   - Language: Swift
   - Include Core Data: Yes

3. Replace the generated files with the files in this repository:
   - Copy all files from `Zitate/` directory into your Xcode project
   - Make sure to add them to the Zitate target

### 2. Configure Project Settings

1. Set **Deployment Target** to iOS 17.0
2. Add **Capabilities**:
   - iCloud → CloudKit
   - Background Modes → Remote notifications
3. Update **Info.plist** with location permission descriptions (already included)

### 3. Build and Run

1. Select an iOS 17.0+ simulator or device
2. Build (⌘B) to compile
3. Run (⌘R) to launch

## Usage

### Creating a Quote

1. Tap the **+** button in the navigation bar
2. Enter your quote text (1-10,000 characters)
3. Location is captured automatically (grant permission when prompted)
4. Tap **Save** to store the entry
